# Metrics | 指标定义

Define each metric before use.
任何指标必须先定义，再使用。

## Recommended Core Metrics | 推荐核心指标

- Task Success Rate.
- Factuality/Error Rate.
- Tool-Use Reliability.
- Latency (p50/p95).
- Cost per successful task.
- Safety violation rate.

- 任务成功率。
- 事实性/错误率。
- 工具调用可靠性。
- 时延（p50/p95）。
- 单次成功任务成本。
- 安全违规率。
