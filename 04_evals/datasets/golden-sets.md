# Golden Sets | 黄金评测集

Document curated, stable evaluation sets.
记录经过筛选且稳定的评测集。

For each set record:

- Set ID and owner.
- Purpose and scope.
- Inclusion/exclusion criteria.
- Redaction/sanitization method.
- Versioning and change log.

每个数据集需记录：

- 数据集 ID 与负责人。
- 目标与范围。
- 纳入/排除标准。
- 脱敏方式。
- 版本与变更记录。
