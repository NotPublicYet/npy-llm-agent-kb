## Summary
<h2>What's changed</h2>
<ul>
<li>Added the <code>/claude-api</code> skill for building applications with the Claude API and Anthropic SDK</li>
<li>Added Ctrl+U on an empty bash prompt (<code>!</code>) to exit bash mode, matching <code>escape</code> and <code>backspace</code></li>
<li>Added numeric keypad support for selecting options in Claude's interview questions (previously only the number row above QWERTY worked)</li>
<li>Added optional name argument to <code>/remote-control</code> and <code>claude remote-control</code> (<code>/remote-control My Project</code> or <code>--name "My Project"</code>) to set a custom session title visible in claude.ai/code</li>
<li>Added Voice STT support for 10 new languages (20 total) — Russian, Polish, Turkish, Dutch, Ukrainian, Greek, Czech, Danish, Swedish, Norwegian</li>
<li>Added effort level display (e.g., "with low effort") to the logo and spinner, making it easier to see which effort setting is active</li>
<li>Added agent name display in terminal title when using <code>claude --agent</code></li>
<li>Added <code>sandbox.enableWeakerNetworkIsolation</code> setting (macOS only) to allow Go programs like <code>gh</code>, <code>gcloud</code>, and <code>terraform</code> to verify TLS certificates when using a custom MITM proxy with <code>httpProxyPort</code></li>
<li>Added <code>includeGitInstructions</code> setting (and <code>CLAUDE_CODE_DISABLE_GIT_INSTRUCTIONS</code> env var) to remove built-in commit and PR workflow instructions from Claude's system prompt</li>
<li>Added <code>/reload-plugins</code> command to activate pending plugin changes without restarting</li>
<li>Added a one-time startup prompt suggesting Claude Code Desktop on macOS and Windows (max 3 showings, dismissible)</li>
<li>Added <code>${CLAUDE_SKILL_DIR}</code> variable for skills to reference their own directory in SKILL.md content</li>
<li>Added <code>InstructionsLoaded</code> hook event that fires when CLAUDE.md or <code>.claude/rules/*.md</code> files are loaded into context</li>
<li>Added <code>agent_id</code> (for subagents) and <code>agent_type</code> (for subagents and <code>--agent</code>) to hook events</li>
<li>Added <code>worktree</code> field to status line hook commands with name, path, branch, and original repo directory when running in a <code>--worktree</code> session</li>
<li>Added <code>pluginTrustMessage</code> in managed settings to append organization-specific context to the plugin trust warning shown before installation</li>
<li>Added policy limit fetching (e.g., remote control restrictions) for Team plan OAuth users, not just Enterprise</li>
<li>Added <code>pathPattern</code> to <code>strictKnownMarketplaces</code> for regex-matching file/directory marketplace sources alongside <code>hostPattern</code> restrictions</li>
<li>Added plugin source type <code>git-subdir</code> to point to a subdirectory within a git repo</li>
<li>Added <code>oauth.authServerMetadataUrl</code> config option for MCP servers to specify a custom OAuth metadata discovery URL when standard discovery fails</li>
<li>Fixed a security issue where nested skill discovery could load skills from gitignored directories like <code>node_modules</code></li>
<li>Fixed trust dialog silently enabling all <code>.mcp.json</code> servers on first run. You'll now see the per-server approval dialog as expected</li>
<li>Fixed <code>claude remote-control</code> crashing immediately on npm installs with "bad option: --sdk-url" (<a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="3986490065" data-permission-text="Title is private" data-url="https://github.com/anthropics/claude-code/issues/28334" data-hovercard-type="issue" data-hovercard-url="/anthropics/claude-code/issues/28334/hovercard" href="https://github.com/anthropics/claude-code/issues/28334">#28334</a>)</li>
<li>Fixed <code>--model claude-opus-4-0</code> and <code>--model claude-opus-4-1</code> resolving to deprecated Opus versions instead of current</li>
<li>Fixed macOS keychain corruption when using multiple OAuth MCP servers. Large OAuth metadata blobs could overflow the <code>security -i</code> stdin buffer, silently leaving stale credentials behind and causing repeated <code>/login</code> prompts.</li>
<li>Fixed <code>.credentials.json</code> losing <code>subscriptionType</code> (showing "Claude API" instead of "Claude Pro"/"Claude Max") when the profile endpoint transiently fails during token refresh (<a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="4013279626" data-permission-text="Title is private" data-url="https://github.com/anthropics/claude-code/issues/30185" data-hovercard-type="issue" data-hovercard-url="/anthropics/claude-code/issues/30185/hovercard" href="https://github.com/anthropics/claude-code/issues/30185">#30185</a>)</li>
<li>Fixed ghost dotfiles (<code>.bashrc</code>, <code>HEAD</code>, etc.) appearing as untracked files in the working directory after sandboxed Bash commands on Linux</li>
<li>Fixed Shift+Enter printing <code>[27;2;13~</code> instead of inserting a newline in Ghostty over SSH</li>
<li>Fixed stash (Ctrl+S) being cleared when submitting a message while Claude is working</li>
<li>Fixed ctrl+o (transcript toggle) freezing for many seconds in long sessions with lots of file edits</li>
<li>Fixed plan mode feedback input not supporting multi-line text entry (backslash+Enter and Shift+Enter now insert newlines)</li>
<li>Fixed cursor not moving down into blank lines at the top of the input box</li>
<li>Fixed <code>/stats</code> crash when transcript files contain entries with missing or malformed timestamps</li>
<li>Fixed a brief hang after a streaming error on long sessions (the transcript was being fully rewritten to drop one line; it is now truncated in place)</li>
<li>Fixed <code>--setting-sources user</code> not blocking dynamically discovered project skills</li>
<li>Fixed duplicate CLAUDE.md, slash commands, agents, and rules when running from a worktree nested inside its main repo (e.g. <code>claude -w</code>)</li>
<li>Fixed plugin Stop/SessionEnd/etc hooks not firing after any <code>/plugin</code> operation</li>
<li>Fixed plugin hooks being silently dropped when two plugins use the same <code>${CLAUDE_PLUGIN_ROOT}/...</code> command template</li>
<li>Fixed memory leak in long-running SDK/CCR sessions where conversation messages were retained unnecessarily</li>
<li>Fixed API 400 errors in forked agents (autocompact, summarization) when resuming sessions that were interrupted mid-tool-batch</li>
<li>Fixed "unexpected tool_use_id found in tool_result blocks" error when resuming conversations that start with an orphaned tool result</li>
<li>Fixed teammates accidentally spawning nested teammates via the Agent tool's <code>name</code> parameter</li>
<li>Fixed <code>CLAUDE_CODE_MAX_OUTPUT_TOKENS</code> being ignored during conversation compaction</li>
<li>Fixed <code>/compact</code> summary rendering as a user bubble in SDK consumers (Claude Code Remote web UI, VSCode extension)</li>
<li>Fixed voice space bar getting stuck after a failed voice activation (module loading race, cold GrowthBook)</li>
<li>Fixed worktree file copy on Windows</li>
<li>Fixed global <code>.claude</code> folder detection on Windows</li>
<li>Fixed symlink bypass where writing new files through a symlinked parent directory could escape the working directory in <code>acceptEdits</code> mode</li>
<li>Fixed sandbox prompting users to approve non-allowed domains when <code>allowManagedDomainsOnly</code> is enabled in managed settings — non-allowed domains are now blocked automatically with no bypass</li>
<li>Fixed interactive tools (e.g., <code>AskUserQuestion</code>) being silently auto-allowed when listed in a skill's allowed-tools, bypassing the permission prompt and running with empty answers</li>
<li>Fixed multi-GB memory spike when committing with large untracked binary files in the working tree</li>
<li>Fixed Escape not interrupting a running turn when the input box has draft text. Use Up arrow to pull queued messages back for editing, or Ctrl+U to clear the input line.</li>
<li>Fixed Android app crash when running local slash commands (<code>/voice</code>, <code>/cost</code>) in Remote Control sessions</li>
<li>Fixed a memory leak where old message array versions accumulated in React Compiler <code>memoCache</code> over long sessions</li>
<li>Fixed a memory leak where REPL render scopes accumulated over long sessions (~35MB over 1000 turns)</li>
<li>Fixed memory retention in in-process teammates where the parent's full conversation history was pinned for the teammate's lifetime, preventing GC after <code>/clear</code> or auto-compact</li>
<li>Fixed a memory leak in interactive mode where hook events could accumulate unboundedly during long sessions</li>
<li>Fixed hang when <code>--mcp-config</code> points to a corrupted file</li>
<li>Fixed slow startup when many skills/plugins are installed</li>
<li>Fixed <code>cd &lt;outside-dir&gt; &amp;&amp; &lt;cmd&gt;</code> permission prompt to surface the chained command instead of only showing "Yes, allow reading from /"</li>
<li>Fixed conditional <code>.claude/rules/*.md</code> files (with <code>paths:</code> frontmatter) and nested CLAUDE.md files not loading in print mode (<code>claude -p</code>)</li>
<li>Fixed <code>/clear</code> not fully clearing all session caches, reducing memory retention in long sessions</li>
<li>Fixed terminal flicker caused by animated elements at the scrollback boundary</li>
