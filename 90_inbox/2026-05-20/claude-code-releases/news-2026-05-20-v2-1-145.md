## Summary
<h2>What's changed</h2>
<ul>
<li>Added <code>claude agents --json</code> to list live Claude sessions as JSON for scripting (tmux-resurrect, status bars, session pickers)</li>
<li>Added <code>agent_id</code> and <code>parent_agent_id</code> attributes to <code>claude_code.tool</code> OTEL spans, and fixed trace parenting so background subagent spans nest under the dispatching Agent tool span</li>
<li>Status line JSON input now includes GitHub repo and PR information when detected</li>
<li><code>/plugin</code> Discover and Browse screens now show a plugin's commands, agents, skills, hooks, and MCP/LSP servers before installation</li>
<li><code>claude agents</code> terminal tab title now shows the awaiting-input count so an alt-tabbed window tells you when an agent needs attention</li>
<li>Slash command and @-mention suggestion list now supports mouse hover and click in fullscreen mode</li>
<li>Stop and SubagentStop hook input now includes <code>background_tasks</code> and <code>session_crons</code> fields</li>
<li>Fixed a permission-prompt bypass where bare variable assignments to non-allowlisted environment variables in Bash commands were auto-approved</li>
<li>Fixed MCP prompt slash commands showing raw server validation errors when a required argument is omitted — the error now names the missing argument and shows expected usage</li>
<li>Fixed the spinner and elapsed-time display freezing until a keypress after the terminal was resized or refocused</li>
<li>Fixed the cross-project resume hint failing in default Windows PowerShell 5.1 — Windows now uses <code>;</code> as the command separator</li>
<li>Fixed voice push-to-talk not working in the agent view's reply pane</li>
<li>Fixed task lists rendering in random order when several tasks are created at once</li>
<li>Fixed stale "Failed to install Anthropic marketplace" banner showing when the marketplace is already installed</li>
<li>Fixed the PR badge in the footer not updating immediately after <code>gh pr create</code> and other PR-state-changing commands run in-session</li>
<li>Fixed Agent Teams teammates with non-ASCII names failing every API call due to invalid header encoding</li>
<li>Fixed <code>/review</code> using a deprecated <code>projectCards</code> GraphQL query that errored on repos with Classic Projects</li>
<li>Fixed <code>claude plugin validate</code> not flagging <code>skills:</code> entries that point at a file instead of a directory — the error now suggests the parent directory</li>
<li>Fixed an infinite loop where a skill using <code>context: fork</code> could repeatedly re-invoke itself instead of running</li>
<li>Improved the Read tool to return a truncated first page with a "PARTIAL view" notice instead of a hard error when a whole-file read exceeds the token limit</li>
</ul>
