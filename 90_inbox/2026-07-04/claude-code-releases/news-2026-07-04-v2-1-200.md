## Summary
<h2>What's changed</h2>
<ul>
<li>Changed <code>AskUserQuestion</code> dialogs to no longer auto-continue by default; opt into an idle timeout via <code>/config</code></li>
<li>Changed the "default" permission mode to "Manual" across the CLI, <code>--help</code>, VS Code, and JetBrains; <code>--permission-mode manual</code> and <code>"defaultMode": "manual"</code> are accepted alongside <code>default</code></li>
<li>Fixed a crash at startup when <code>disabledMcpServers</code> or <code>enabledMcpServers</code> in <code>.claude.json</code> is set to a non-array value</li>
<li>Fixed background sessions silently stopping mid-turn after sleep/wake or when reopening a stalled session</li>
<li>Fixed background sessions re-running a turn cancelled with Esc after a stall respawn</li>
<li>Fixed background agents never starting again after a crash left a stale <code>daemon.lock</code> whose PID the OS reused</li>
<li>Fixed background-agent daemon handover so a reinstalled older build can no longer take over the daemon; build recency is now judged by the version's embedded build timestamp</li>
<li>Fixed background-agent roster issues: transient corruption permanently disabling orphan cleanup, older binaries not preserving fields written by newer versions, and socket auth tokens being stripped during daemon restarts</li>
<li>Fixed subagents cut off by a rate limit before producing any text output returning an empty result instead of failing cleanly</li>
<li>Fixed control bytes from background-agent output reaching the terminal in the agent view</li>
<li>Fixed <code>claude agents --plugin-dir &lt;dir&gt;</code> not showing the plugin's agents and skills in the agent view when the flag is placed after <code>agents</code></li>
<li>Fixed project-scoped plugins not loading correctly from git worktrees of the same repository</li>
<li>Fixed <code>/mcp</code> server list not tracking focus for screen readers and magnifiers</li>
<li>Fixed voice dictation showing a misleading "Voice connection failed" message when a recording captures no audio</li>
<li>Fixed rendering flicker under tmux 3.4+ by enabling synchronized terminal output</li>
<li>Improved screen-reader output: decorative glyphs are now hidden, transcript symbols read as short labels, and nested tables read as <code>Header: value.</code> lines</li>
<li>Improved the install script to explain when installation is killed by the system running out of memory</li>
</ul>
