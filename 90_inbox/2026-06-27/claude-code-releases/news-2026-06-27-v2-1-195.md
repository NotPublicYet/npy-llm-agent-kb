## Summary
<h2>What's changed</h2>
<ul>
<li>Added <code>CLAUDE_CODE_DISABLE_MOUSE_CLICKS</code> to disable mouse click/drag/hover in fullscreen mode while keeping wheel scroll</li>
<li>Fixed hook matchers with hyphenated identifiers (e.g. <code>code-reviewer</code>, <code>mcp__brave-search</code>) accidentally substring-matching — they now exact-match. Use <code>mcp__brave-search__.*</code> to match all tools from a hyphenated MCP server.</li>
<li>Fixed voice dictation on macOS capturing silence in long-running sessions after the default input device changes</li>
<li>Fixed voice dictation auto-submit never firing for languages written without spaces (Japanese, Chinese, Thai)</li>
<li>Fixed external plugins enabled only by project <code>.claude/settings.json</code> not requiring explicit install consent on every loader path</li>
<li>Fixed <code>/plugin</code> Enable/Disable not working when a plugin's <code>plugin.json</code> <code>name</code> differs from its marketplace entry name</li>
<li>Fixed background jobs disappearing from <code>claude agents</code> or losing data when written by a newer Claude Code version</li>
<li>Fixed reopening a crashed background task showing a blank screen for up to 5 seconds instead of its restart</li>
<li>Fixed background agent daemons running unreachable when the control socket fails to start, blocking restarts</li>
<li>Improved voice mode on Linux: now distinguishes "no microphone" from "SoX not installed" when SoX is present but no audio capture device exists</li>
<li>Improved <code>claude agents</code> completed list to fill available vertical space; on short terminals the header compacts so live sessions stay visible</li>
<li>Improved Remote session startup with a provisioning checklist while the container starts</li>
</ul>
