# Inbox | 收件箱

`90_inbox/` is a temporary landing zone for untriaged content.
`90_inbox/` 是待分诊内容的临时落地区。

## Folder Layout | 目录结构

News auto-ingest files are organized as:
新闻自动抓取文件按如下结构组织：

- `90_inbox/YYYY-MM-DD/<source-id>/news-YYYY-MM-DD-short-title.md`

Example:
示例：

- `90_inbox/2026-02-22/openai-blog/news-2026-02-22-our-first-proof-submissions.md`

## Hard Rule | 强制规则

Anything in inbox must be triaged within 7 days.
收件箱内任何条目必须在 7 天内完成分诊。

## Triage Outcomes | 分诊结果

- Move to the correct permanent folder.
- Request missing context from owner.
- Archive as non-actionable with rationale.

- 移动到正确的正式目录。
- 向提交人补充必要上下文。
- 标记为不可执行并附理由后归档。

## Required Metadata | 必要元数据

- Submitted date
- Submitter
- Proposed destination
- Sensitivity notes

- 提交日期
- 提交人
- 目标目录
- 敏感性备注
