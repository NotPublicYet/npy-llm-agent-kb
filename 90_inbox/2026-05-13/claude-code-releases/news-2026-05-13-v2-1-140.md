## Summary
<h2>What's changed</h2>
<ul>
<li>Improved Agent tool <code>subagent_type</code> matching to accept case- and separator-insensitive values (e.g. <code>"Code Reviewer"</code> resolves to <code>code-reviewer</code>)</li>
<li>Updated agent color palette</li>
<li>Fixed <code>/goal</code> silently hanging when <code>disableAllHooks</code> or <code>allowManagedHooksOnly</code> is set — now shows a clear message instead of an indicator that never resolves</li>
<li>Fixed a regression in settings hot-reload where symlinked settings files caused misattributed change events and spurious <code>ConfigChange</code> hooks</li>
<li>Fixed <code>claude --bg</code> failing with "connection dropped mid-request" when the background service was about to idle-exit</li>
<li>Fixed background service startup failing on machines with enterprise endpoint security by allowing more time</li>
<li>Fixed remote managed settings not retrying on 401 — now retries once with a force-refreshed token</li>
<li>Fixed managed <code>extraKnownMarketplaces</code> auto-update policy not being persisted to <code>known_marketplaces.json</code></li>
<li>Fixed <code>/loop</code> scheduling redundant wakeups to poll for background tasks that already notify on completion</li>
<li>Fixed a recurring event-loop stall on Windows when a missing executable (e.g. <code>gh</code>) triggered synchronous <code>where.exe</code> re-spawns on every check</li>
<li>Fixed <code>Read</code> tool calls failing validation when <code>offset</code> is passed as a whitespace-padded or <code>+</code>-prefixed string</li>
<li>Fixed native terminal cursor not staying at the input caret when the terminal loses focus</li>
<li>Plugins now warn when a default component folder (e.g. <code>commands/</code>) is silently ignored because <code>plugin.json</code> sets the matching key. Shown in <code>/doctor</code>, <code>claude plugin list</code>, and <code>/plugin</code>.</li>
</ul>
